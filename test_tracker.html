✅ Checklist de Validação no Banco

Após testar, vá no Supabase → Table Editor e verifique:

  • lt_leads → Novo registro com seu lt_id
  • lt_leads → fbp e fbc preenchidos (se Pixel FB ativo)
  • lt_leads → first_utm_source = "test" (da URL)
  • lt_visits → Registro com page_url desta página
  • lt_events → Evento pageview
  • lt_events → Evento scroll (25%, 50%, etc)
  • lt_events → Evento click (btn-test-cta)
  • lt_events → Evento form_submit (com email no metadata)
  • lt_leads → email preenchido após form_submit
  • lt_events → Evento begin_checkout
